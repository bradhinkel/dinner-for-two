03 · Evening · interactions08

Course swap

Per-course regenerate. Oxblood ↻ glyph in a 30×30 round 1px-rule outline.

  • Tap → row drops to 45% opacity, “…” badge appears at right.
  • 1.1s minimum; await streaming response.
  • On settle → row returns to 100% with a small “NEW” oxblood badge for 3s.
  • Sibling rows untouched. Pairing untouched unless the swap changes the entrée in two-entrée mode (then re-pair that entrée only).

Model toggle

Two-cell segmented in the “COMPOSED AS” strip.

  • Switching mode triggers a full re-compose for the same restaurant.
  • Shows the composing log inline above the menu (collapsed version, 4 rows max).
  • Pairing strategy flips: shared → one bottle; two-entrée → glass-by-glass.

Save

Top-right “Save · Share”.

  • Save mints a share_token server-side and copies the URL.
  • Toast at bottom: italic “link copied · expires in 6h” on ink bg, 2.5s.
  • No accounts required.

Reserve / Directions

  • Reserve: window.open(reservationUrl). New tab.
  • Directions: deep-link maps:?q=address on iOS, geo: on Android, otherwise Google Maps URL.
  • Neither leaves an in-app modal; both are external handoffs.

Data shape

type Evening = {
  id: string,
  restaurant: { name, neighborhood, cuisine, priceTier, vibe[], blurb,
                address, walkTime, verifiedAt, reservationUrl, lat, lng },
  rationale: string,
  estimated: number,        // dollars, two-person estimate
  estimatedRange: string,   // "$168–$212"
  model: "shared" | "two-entree",
  courses: [{ type, name, desc, price, note }],
  pairing: { title, name, region, style, bottlePrice, note }
}

Match scoring

Confidence from semantic match × constraint fit. Show as integer percent. Never below 70 — if no candidate clears, show empty state.

Voice · the one-line reason

Short, observational, specific. Always under 14 words. Examples in reference.

card.reason · examples The courtyard. Made for anniversaries. Mole that takes three days.

Brick room behind a leather curtain. Italian list, Mexican kitchen.

Quieter, warmer, less of a scene. Ceviche worth the trip.

  • Tap CTA → cross-fade to 02 Composing (mode = re-compose), then 03 Evening.
  • Brief is unchanged; restaurant ID is overridden.

06 · Deferred screens11

Resolve in code.

Two screens are intentionally not laid out. Both are simple enough that the design system carries them. Build them with the existing tokens and voice.

A · No-match / error Deferred

Triggered when the matcher returns no rooms ≥70% confidence, or the LLM call fails.

Layout sketch

  • Use 02 Composing chrome.
  • Replace hero with: italic display 36px, “Couldn't find a fit.” (oxblood last word).
  • Italic body: explain what didn't match (e.g. “No room in our set is open tonight under $80 per person, in a romantic register”).
  • Two CTAs: oxblood “WIDEN THE BRIEF” (returns to 01 with a hint), ink-outline “TRY ANOTHER NIGHT”.

Voice

“Couldn't find a fit. Tonight is tight on rooms — most of our $$ Mexican spots are dark on Sundays. Widen the price, or pick a different night.”

Never: apologize, blame the user, or surface technical errors.

B · Reservation handoff confirmation Deferred

Brief screen shown for ~1.5s after tapping Reserve, before the OpenTable / Resy link opens.

Layout sketch

  • Centered on paper. Wordmark top.
  • Italic display 32px: “Sending you to OpenTable.”
  • Italic body: party of two, the date, the restaurant.
  • Hairline + small spinner-replacement: a gentle 3-dot fade-cycle (no spin).
  • Auto-redirect at 1.5s; manual “Open now →” underlined link.

Why a moment

The handoff to a third-party booking flow is jarring. A brief in-voice transition lets the user breathe before the change of register, and reinforces that we made the recommendation.

✦

For anything else not specified — toasts, modals, micro-copy — reach for italic Cormorant, hairlines, and oxblood used sparingly. Less, said well.

Endnote∞

Build it
quietly.

Honor the typographic restraint. Avoid spinners. Avoid emoji. Avoid drop shadows on UI. Let the hero photography and italic display do the heavy lifting; the rest is hairlines, labels, and well-chosen words.

When in doubt, do less.
